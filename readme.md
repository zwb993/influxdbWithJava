### 测试timescaledb 性能测试步骤

1. 下载postgresql 12，[下载链接](https://www.enterprisedb.com/downloads/postgres-postgresql-downloads)， 并安装
2. 将安装目录的lib和bin路径加入path中
3. 下载timescaledb,[下载链接](https://timescalereleases.blob.core.windows.net/windows/timescaledb-postgresql-12_1.7.1-windows-amd64.zip) 解压，将压缩包中的timescaledb.dll放置在postgresql的lib目录下，运行setup.exe
4. 编译运行jar包
